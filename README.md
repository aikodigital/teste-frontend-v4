# Gestão de Equipamentos

## 1 - Para esse projeto, foram utilizadas as seguintes versões:

	Node.js - 18.17.0
	React - 18
	Next.js - 14.0.4


## 2 - Instalação e execução do projeto;

Instalação de dependências:

    npm install
	
Executar projeto no modo de desenvolvimento:
	
    npm run dev
